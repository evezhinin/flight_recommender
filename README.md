# Flight Recommender System
Big Data Systems — NYIT Spring 2026

## Setup
pip3 install -r requirements.txt

## Project Structure
See the implementation guide for full details.

### 1. update your branch
git pull
### 2. Create virtual environment (Python 3.12 required)
```bash
/Library/Frameworks/Python.framework/Versions/3.12/bin/python3.12 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### 3. Download datasets
Download these files and place them in the `data/` folder:

| Mendeley Flight Delays | data.mendeley.com/datasets/j3zfnxk56h/1 → input_data.csv | `data/mendeley_delays.csv` |
| Airline Passenger Satisfaction | kaggle.com/datasets/teejmahal20/airline-passenger-satisfaction | `data/satisfaction.csv` |

### 4. Run preprocessing (Google Colab recommended for memory)
Upload datasets to Google Drive and run `scripts/preprocess.py` in Colab.
Download the output `unified_flights.csv` into your `data/` folder.

### 5. Run the MapReduce pipeline
```bash
python3 pipeline.py
```

Expected output:
Running pipeline on data/unified_flights.csv...
jobs/job1_clean.py: 33,858 lines in ~7s
jobs/job2_score.py: 33,858 lines in ~5s
jobs/job3_topk.py:  13,939 lines in ~8s
Pipeline complete in ~20s


### 6. Query recommendations
```bash
python3 scripts/query_interface.py JFK LAX
python3 scripts/query_interface.py LGA ROC
```

### 7. Run with custom weights
```bash
# Budget traveler (price-heavy)
python3 pipeline.py 0.55 0.15 0.10 0.15 0.05

# Business traveler (speed + reliability)
python3 pipeline.py 0.15 0.30 0.25 0.20 0.10
```

## Scoring Model
| Factor            | Default Weight | Description 

| Price             | 35% | Cheaper = higher score 
| Duration          | 20% | Shorter = higher score 
| Stops             | 15% | Direct = 1.0, each stop reduces score 
| Delay Probability | 20% | Historical on-time rate per route 
| Airline Quality   | 10% | Carrier reliability score from BTS data
